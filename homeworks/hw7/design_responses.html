Emergence is where we can see the outlines of things quickly, reification is when we can see things even if they are incomplete, multi-stability is where we see confusing things in multiple ways, and invariance is when we can see simple things regardless of their rotation. These apply to UI design as they can draw emphasis to certain parts of a website along with make the website overall easier to look at.
